## Hypothesis Testing Framework

Each campaign tests hypotheses:

```
HYPOTHESIS 1:
"CTO title converts better than VP Engineering"

TEST:
- Send variant A to 50 CTOs
- Send variant B to 50 VPs
- Compare reply rates

RESULT:
CTO: 45% reply rate ✓ WINNER
VP: 28% reply rate

LEARNING:
Target CTOs first, VPs second

ACTION:
Next campaign prioritizes CTO search in Parallel
```
